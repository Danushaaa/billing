Help Section - Cashier System

Guide for New Users

Welcome to the Cashier System

Thank you for using the Cashier System! This guide will walk you through the steps to place customer orders, check item availability, manage quantities, and generate bills efficiently. Follow these instructions to get started and make the most of the system.

Getting Started

1. Access the Interface

Log in to the system through the admin dashboard (e.g., admin-dashboard.html) using your assigned credentials. Navigate to the "Cashier Interface - Place Order" page by selecting the "Place Order" option.

2. Interface Overview

The interface includes fields for customer details, item selection, quantity input, and an order summary table. It automatically checks item existence and stock levels, calculates bills, and provides notifications for any issues.

Step-by-Step Guide to Using the System

1. Enter Customer Information

In the "Customer Account Number" field, type the unique account number of the customer (e.g., "C001"). This ensures the order is linked to the correct customer record.

2. Add Items to the Order

Enter the "Item Code" (e.g., "B001") in the designated field. Input the desired "Quantity" (minimum 1) and click "Add Item" to add it to the order.

  • Item Check: The system verifies if the item exists. If not, a notification will pop up (e.g., "Item not found: B999").
  • Quantity Check: The system checks available stock. If the requested quantity exceeds stock, a notification will display the available amount (e.g., "Insufficient stock for B001. Available: 50").
3. Review and Manage Order

The "Order Items" table displays all added items (Item Code, Quantity, Unit Price, Subtotal). Use the "Remove" button to delete items if needed. The "Total Amount" updates automatically.

4. Submit the Order and Generate Bill

Click "Submit Order" to process the order, update stock, and save it. If successful, a bill with the total amount (e.g., "Total Amount: $21.00") appears, printable via the "Print Bill" button. If thereâ€™s an error, a notification will guide you to resolve it.

Key Features

  • Real-Time Validation: Checks item existence and stock levels before adding to the order.
  • Automatic Billing: Calculates the total amount based on unit prices and quantities.
  • Notifications: Alerts you to issues like unavailable items or insufficient stock.
  • Order History: Orders are saved in the database for future reference.

Troubleshooting Tips

"Item not found" Notification: Verify the item code. If correct, ask the admin to add the item to the database.

"Insufficient stock" Notification: Reduce the quantity or inform the customer. The admin can restock items via the item management section.

Submission Fails: Ensure all fields are completed and try again. Check your internet connection if the issue persists, and contact support if needed.

Best Practices

  • Always confirm the customer account number before proceeding.
  • Check item availability and stock before committing to an order.
  • Review the order summary carefully before submission.
  • Save or print the bill for customer records after each transaction.

Additional Support

For further assistance, explore the help section on the admin dashboard or reach out to your system administrator. Keep the item and customer databases updated to ensure smooth operation.
